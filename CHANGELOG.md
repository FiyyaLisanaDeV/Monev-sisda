# Changelog

Semua perubahan penting pada repository ini dicatat di file ini.

## [Unreleased]

### Added

- Root `README.md` untuk orientasi repo dan cara menjalankan aplikasi.
- `dashboardproject.md` sebagai catatan progres kerja terbaru.
- Narasi evaluasi visual mobile di `narasi_kesalahan_visual_dashboard_mobile.md`.

### Changed

- Dashboard Filament diselaraskan dengan referensi visual di folder `Dashboard Design`.
- Panel admin dikunci ke light mode.
- Detail paket berisiko diubah menjadi modal/slide-over yang lebih ramah mobile.
- Kolom `Status` pada tabel paket berisiko dipindah ke posisi paling depan.

### Fixed

- Konten detail paket yang sebelumnya terpotong di mobile.
- Tampilan detail paket yang terlalu bergantung pada form default Filament.

## [0.1.0] - 2026-06-02

### Added

- Inisialisasi repository dashboard Laravel + Filament.
- Dashboard eksekutif, analisis paket, analisis lokasi, perbandingan satker, dan paket berisiko.
- Custom styling premium light mode.
- Dokumentasi referensi desain dan proyek.
